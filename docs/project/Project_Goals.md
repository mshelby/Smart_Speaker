# Project Goals

A premium, fully local smart speaker designed as a spiritual successor to the Amazon Echo 4th Gen, built with superior acoustics, thoughtful craftsmanship, and complete privacy through Home Assistant integration.

**Key Objectives:**
- Target realistic internal acoustic volume of 2.8 – 2.95 liters
- Premium feel through substantial weight (~260–290g in base) and refined details
- Modular three-part design for serviceability
- Timeless minimalist aesthetics with intentional 1 mm shadow gap
- Honest engineering that balances acoustics and premium tactile feel

**The Mark Shelby Design Philosophy**  
Inspired by refined minimalism and functional clarity, this project prioritizes honest engineering, acoustic performance, and timeless aesthetics while remaining fully local and privacy-focused.
